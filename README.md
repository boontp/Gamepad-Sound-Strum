# Gamepad-Sound-Strum
Gamepad Sound Strum

It work in progress
https://boontp.github.io/Gamepad-Sound-Strum/

now it have only 4 line bass mode

use left analog to play fret
use right button to play string 1234
use L1/R1/L2/R2 to modify fret

in PC can use WASD as analog
use HJKL for play string
use CVBN to modify fret

support controller
Piano part and Fret part is for demonstate key only. not for press
